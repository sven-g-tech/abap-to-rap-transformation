# Transformation Approach for Modernizing Classical ABAP Applications into RAP-based Architectures

## Repository Structure
```
prototypes/
├── batch-job/
├── dynpro-report/

transformation-approach/
├── transformation-matrix.md
└── transformation-process.pdf
```

## Description

- [`prototypes/`](./prototypes/)  
  Contains the prototypical applications used for the iterative development and evaluation of the transformation approach.

- [`prototypes/batch-job/`](./prototypes/batch-job/)  
  Backend-oriented prototype, including the implementation and UML diagrams (classical and RAP-based).

- [`prototypes/dynpro-report/`](./prototypes/dynpro-report/)  
  UI-oriented prototype, including the implementation and UML diagrams (classical and RAP-based).

- [`transformation-approach/`](./transformation-approach/)  
  Contains the transformation matrix (mapping between classical SAP artifacts and RAP artifacts) and the transformation process (procedure for the systematic application of the matrix).
